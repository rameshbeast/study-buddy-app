## Project Description
The Study Buddy App is a collaborative study support platform designed to help university students organize their learning activities and work together effectively.

The system allows students to:
- Create and manage study groups
- Share notes and learning resources
- Set study goals and track progress
- Schedule meetings and reminders
- Communicate with group members through discussion

The goal of the application is to improve productivity, motivation, and teamwork among students by providing a centralized environment for academic collaboration.
